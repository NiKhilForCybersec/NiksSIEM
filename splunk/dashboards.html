Dashboards

Building security dashboards with SimpleXML and Dashboard Studio

Dashboard Types

TypeFormatUse Case
Classic (SimpleXML)XMLQuick dashboards, existing content
Dashboard StudioJSONModern layouts, advanced visualizations

SimpleXML Example

XML Security Overview Dashboard 
<dashboard>
  <label>Security Overview</label>
  <row>
    <panel>
      <title>Failed Logins (24h)</title>
      <single>
        <search>
          <query>| tstats count from datamodel=Authentication where Authentication.action=failure</query>
          <earliest>-24h</earliest>
        </search>
      </single>
    </panel>
    <panel>
      <title>Open Notables</title>
      <single>
        <search>
          <query>\`notable\` | search status_label="New" | stats count</query>
        </search>
      </single>
    </panel>
  </row>
  <row>
    <panel>
      <title>Top Attackers</title>
      <table>
        <search>
          <query>| tstats count from datamodel=Authentication where Authentication.action=failure by Authentication.src | sort -count | head 10</query>
        </search>
      </table>
    </panel>
  </row>
</dashboard>
