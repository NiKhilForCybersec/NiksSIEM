Splunk SOAR

Splunk SOAR (formerly Phantom) provides security orchestration, automation, and response. Automate repetitive tasks, orchestrate tools, and respond to threats faster.

SOAR Core Concepts

Playbooks

Automated workflows that orchestrate actions across tools. Visual or code-based.

Apps

Connectors to security tools (firewalls, EDR, ticketing). 300+ available.

Containers

Represent incidents/events. Hold artifacts, notes, actions taken.

Artifacts

IOCs within containers (IPs, hashes, domains). Can be enriched/actioned.

Interview Questions

Good automation candidates:

  • Repetitive tasks done 10+ times/day
  • Well-defined decision trees
  • Tasks requiring multiple tool access
  • Time-sensitive responses

Keep human-in-the-loop for:

  • High-impact actions (disable exec account)
  • Ambiguous situations needing judgment
  • Actions with business impact
