10%

Precedence (Highest → Lowest)

  1. $SPLUNK_HOME/etc/system/local/ ← Highest
  2. $SPLUNK_HOME/etc/apps/<app>/local/
  3. $SPLUNK_HOME/etc/apps/<app>/default/
  4. $SPLUNK_HOME/etc/system/default/ ← Lowest

Admin Exam Summary

  • Focus: Indexes (15%), Auth (10%), Parsing (10%), Forwarders (10%), Config (10%)
  • Required paths: homePath, coldPath, thawedPath
  • frozenTimePeriodInSecs: 90 days = 7776000
  • License violation: Search disabled, indexing continues
  • btool: Your best friend for troubleshooting configs
