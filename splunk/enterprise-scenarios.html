Interview Questions

  1. Architecture: Multi-site indexer cluster with 40-50 indexers across 2-3 sites
  2. Search tier: Search head cluster with 10+ members, load balanced
  3. Data routing: Heavy forwarders at each site for aggregation
  4. Replication: RF=3 (site1=2, site2=1), SF=2 for search availability
  5. Storage: SmartStore with S3/Azure Blob for cost optimization
  6. License: Enterprise license with infrastructure pool
