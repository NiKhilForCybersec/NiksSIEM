Splunk Platform

Splunk is a leading data analytics platform widely used for security monitoring (SIEM), IT operations, and business intelligence. Enterprise Security (ES) transforms Splunk into a full-featured SIEM.

Splunk Products

Splunk Enterprise

Core platform for data ingestion, indexing, and search. Foundation for all Splunk solutions.

Enterprise Security

Premium SIEM app. Correlation searches, notable events, investigations, RBA.

SOAR

Security orchestration and automation. Playbooks, case management, integrations.

Splunk Cloud

Cloud-hosted Splunk. Managed infrastructure, automatic updates.

SPL Overview

Search Processing Language

SPL is Splunk's query language. It uses a pipe-based syntax where each command transforms data and passes it to the next. Master SPL for effective threat hunting, detection engineering, and analysis.

Basic SPL Structure
index=security sourcetype=WinEventLog:Security EventCode=4625
| stats count by Account_Name, Source_Network_Address
| where count > 10
| sort - count
| head 20

Interview Questions

  • Schema-on-read: No predefined schema needed - index any data
  • SPL flexibility: Powerful query language for any analysis
  • Horizontal scaling: Add indexers for more capacity
  • App ecosystem: Thousands of apps on Splunkbase
  • Beyond security: Same platform for IT ops, business analytics

Trade-offs: License cost (per GB), complexity for large deployments
