Splunk Enterprise Security

Splunk is the industry-leading SIEM platform known for its unmatched flexibility, powerful SPL query language, and extensive app ecosystem. Deploy on-prem, cloud, or hybrid for complete security operations.

Prerequisites

Linux/Windows administration
Basic networking
Security operations concepts
Log analysis experience

Core Components

Indexers

Store data, process searches. Scale horizontally in clusters for capacity and HA.

Search Heads

User interface, run searches, host apps/dashboards. Cluster for availability.

Forwarders

Universal (lightweight) or Heavy (parsing). Collect and forward data.

Enterprise Security

Premium security app with correlation searches, notable events, RBA.

Interview Questions

Data Flow:

  1. Collection: Universal Forwarders on endpoints
  2. Processing: Heavy Forwarders parse/filter/route
  3. Indexing: Indexers store and search data
  4. Search: Search Heads provide UI

Management: Deployment Server (configs), Cluster Master (replication), License Master (usage)

Universal: Lightweight (~100MB), forwards only, no parsing. Deploy everywhere.

Heavy: Full Splunk instance, parses data, can filter/route/enrich. Use for syslog aggregation.

Common Information Model standardizes field names across sources. TAs map vendor fields to CIM. ES requires CIM compliance for correlation searches.

Key models: Authentication (user, src, dest, action), Network Traffic (src_ip, dest_ip, bytes), Endpoint (process, parent, cmdline)
