Host Management

Manage endpoints, groups, and policies in CrowdStrike Falcon.

Host Groups

Dynamic Groups

Automatically group hosts based on properties:

  • Operating system type and version
  • Domain membership
  • OU (Organizational Unit)
  • Tags and custom properties

Static Groups

Manually assign hosts to groups for specific policy application.

Prevention Policies

Policy Settings

  • Detection: Alert only, no blocking
  • Prevention: Block malicious activity
  • Aggressive: Maximum protection, may impact performance
