Event Search Quick Reference

Search Syntax

Operator Description Example
= Exact match ImageFileName="powershell.exe"
!= Not equal UserName!="SYSTEM"
=* Wildcard match CommandLine=*"-enc"*
> < Comparison BytesSent>1000000
AND Both conditions ProcessStartTime>-1h AND ImageFileName="cmd.exe"
OR Either condition ImageFileName="psexec.exe" OR ImageFileName="paexec.exe"

Essential Hunting Queries

// Encoded PowerShell Commands
event_simpleName=ProcessRollup2 
| ImageFileName="*powershell.exe" 
| CommandLine=*"-enc"* OR CommandLine=*"-encodedcommand"*
// Suspicious Parent-Child Relationships (Office spawning cmd/PowerShell)
event_simpleName=ProcessRollup2 
| ParentBaseFileName IN ("winword.exe", "excel.exe", "outlook.exe") 
| ImageFileName IN ("cmd.exe", "powershell.exe", "wscript.exe", "cscript.exe")
// Lateral Movement Tools
event_simpleName=ProcessRollup2 
| ImageFileName IN ("psexec.exe", "psexesvc.exe", "wmic.exe", "winrm.exe", "wmiprvse.exe")
// LSASS Access Detection
event_simpleName=ProcessRollup2 
| TargetFileName="*lsass.exe" 
| ParentBaseFileName!="svchost.exe"
// Scheduled Task Creation
event_simpleName=ProcessRollup2 
| ImageFileName="schtasks.exe" 
| CommandLine=*"/create"*

rm [path]       // Delete file
memdump [PID]   // Memory dump
getsid          // Get SID
zip [src] [dst] // Archive files
put [local] [remote]  // Upload file
get [remote]    // Download file

// Network Contain (isolate host)
