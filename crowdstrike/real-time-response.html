Real Time Response (RTR)

Use CrowdStrike Real Time Response for live endpoint investigation and remediation.

RTR Overview

Real Time Response provides a secure shell-like interface to endpoints for live investigation and remediation.

Permission Levels

  • Read-Only: View system state, run queries
  • Responder: Basic remediation actions
  • Admin: Full system access including scripts

Common Commands

RTR Commands
# List running processes
ps

# Get file information
filehash [path]

# List directory contents
ls [path]

# Get registry value
reg query [key]

# Kill process
kill [pid]

# Network connections
netstat
