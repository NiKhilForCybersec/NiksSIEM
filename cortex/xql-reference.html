XQL Reference

Complete XDR Query Language reference with syntax, operators, functions, and production-ready examples

Query Structure

╔══════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════╗
║                                              XQL QUERY STRUCTURE                                                                      ║
╠══════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════╣
║                                                                                                                                       ║
║   ┌──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐   ║
║   │  dataset = xdr_data                         // 1. DATA SOURCE - Required, specifies the dataset to query                    │   ║
║   │  | preset = xdr_process                     // 2. PRESET (optional) - Pre-filtered views for common use cases               │   ║
║   │  | filter event_type = PROCESS              // 3. FILTER - Narrow down results (WHERE clause equivalent)                    │   ║
║   │  | filter action_process_image_name contains "powershell"                                                                    │   ║
║   │  | fields agent_hostname, _time, ...        // 4. FIELDS - Select specific columns to return                                │   ║
║   │  | alter new_field = concat(a, b)           // 5. ALTER - Create calculated fields                                          │   ║
║   │  | comp count() as total by hostname        // 6. COMP - Aggregate/compute statistics (GROUP BY equivalent)                 │   ║
║   │  | sort desc _time                          // 7. SORT - Order results                                                      │   ║
║   │  | limit 100                                // 8. LIMIT - Restrict number of results                                        │   ║
║   └──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘   ║
║                                                                                                                                       ║
╚══════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════╝

Dataset Description Key Fields

xdr_event_log event_type = EVENT_LOG Windows event logs

Filter Operators

Comparison Operators
= , != , < , > , <= , >=
// Examples
| filter severity = "High"
| filter _time >= timestamp("2024-01-01")
| filter bytes_sent > 1000000
String Operators
contains, ~contains (case-insensitive)
!contains, !~contains
in, not in
// Examples
| filter cmd contains "-enc"
| filter process ~contains "powershell"
| filter user in ("admin", "system")
Pattern Matching
matches (regex)
// Examples
| filter filename matches ".*\.exe$"
| filter ip matches "^10\.\d+\.\d+\.\d+$"
| filter path matches ".*\\Temp\\.*"
Logical Operators
and, or, not
() for grouping
// Examples
| filter (severity = "High" or severity = "Critical")
| filter user != null and auth_result = "Failed"
| filter not (ip in allow_list)

Functions Reference

String Functions

Function Description Example
lowercase(str) Convert to lowercase alter user_lower = lowercase(user_name)
uppercase(str) Convert to uppercase alter host_upper = uppercase(hostname)
concat(str1, str2, ...) Concatenate strings alter full_path = concat(dir, "\\", filename)
substring(str, start, len) Extract substring alter short = substring(hash, 0, 8)

replace(str, old, new) Replace substring alter clean = replace(cmd, " ", " ")
trim(str) Remove whitespace alter trimmed = trim(input)
length(str) String length alter cmd_len = length(command_line)

Function Description Example

Production Threat Hunting Queries

Credential Access Detection

// LSASS Access - Credential Dumping Detection
dataset = xdr_data
| filter event_type = PROCESS
| filter action_process_image_name ~contains "lsass"
| filter actor_process_image_name not in ("svchost.exe", "wininit.exe", "csrss.exe", "services.exe")
| fields _time, agent_hostname, actor_process_image_name, actor_process_image_path, 
         action_process_image_name, actor_process_image_command_line
| sort desc _time

Encoded PowerShell Detection

// Detect Base64 Encoded PowerShell Commands
dataset = xdr_data
| preset = xdr_process
| filter action_process_image_name ~contains "powershell"
| filter action_process_image_command_line contains "-enc" 
         or action_process_image_command_line contains "-encodedcommand"
         or action_process_image_command_line contains "frombase64"
| fields _time, agent_hostname, actor_process_image_name, action_process_image_command_line
| alter cmd_length = length(action_process_image_command_line)
| filter cmd_length > 500  // Long encoded commands are suspicious
| sort desc _time

Lateral Movement Detection

// Detect Remote Execution Tools
dataset = xdr_data
| preset = xdr_process
| filter action_process_image_name in ("psexec.exe", "psexesvc.exe", "wmic.exe", 
         "winrs.exe", "wmiexec.exe", "smbexec.exe", "atexec.exe")
| fields _time, agent_hostname, actor_process_image_name, actor_process_image_path,
         action_process_image_name, action_process_image_command_line
| sort desc _time

Persistence Mechanisms

// Registry Run Key Modifications
dataset = xdr_data
| preset = xdr_registry
| filter action_registry_key_name contains "\\Software\\Microsoft\\Windows\\CurrentVersion\\Run"
| filter action_registry_value_name != null
| fields _time, agent_hostname, actor_process_image_name, action_registry_key_name,
         action_registry_value_name, action_registry_data
| sort desc _time
// Scheduled Task Creation
dataset = xdr_data
| preset = xdr_process
| filter action_process_image_name = "schtasks.exe"
| filter action_process_image_command_line contains "/create"
| fields _time, agent_hostname, actor_process_image_name, action_process_image_command_line
| sort desc _time

Data Exfiltration Indicators

// Large Outbound Transfers
dataset = xdr_data
| preset = xdr_network
| filter action_remote_ip != null
| filter action_local_ip matches "^(10\\.|172\\.(1[6-9]|2[0-9]|3[01])\\.|192\\.168\\.)"
| filter action_remote_ip not matches "^(10\\.|172\\.(1[6-9]|2[0-9]|3[01])\\.|192\\.168\\.)"
| comp sum(action_total_sent_bytes) as bytes_sent, count() as conn_count by agent_hostname, action_remote_ip
| filter bytes_sent > 100000000  // >100MB
| sort desc bytes_sent

DNS Tunneling Detection

// Detect Potential DNS Exfiltration
dataset = xdr_data
| preset = xdr_network
| filter action_app_protocol = "DNS"
| comp count() as query_count, count_distinct(action_external_hostname) as unique_domains by agent_hostname
| filter query_count > 1000 and unique_domains > 500
| sort desc query_count

Interview Preparation

How would you hunt for credential dumping using Cortex XDR?

I would use XQL to query for LSASS access patterns:

First, I'd look for any process accessing LSASS that isn't a known legitimate process like svchost.exe or csrss.exe. In XQL:

dataset = xdr_data | filter event_type = PROCESS | filter action_process_image_name ~contains "lsass" | filter actor_process_image_name not in ("svchost.exe", "wininit.exe")

I'd also look for Mimikatz-related activity by searching for suspicious command-line patterns like "sekurlsa" or known Mimikatz process names. Additionally, I'd check for process injection events targeting LSASS using the xdr_injection preset.

The key indicators are: unusual parent processes accessing LSASS, command lines containing credential dumping keywords, and any remote thread creation into LSASS.
