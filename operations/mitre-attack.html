MITRE ATT&CK Framework

Using MITRE ATT&CK for threat modeling, detection engineering, and incident analysis.

Framework Overview

Matrix Structure

  • Tactics: The "why" - adversary goals (14 tactics)
  • Techniques: The "how" - methods to achieve goals
  • Sub-techniques: Specific variations of techniques
  • Procedures: Real-world implementations by threat actors
