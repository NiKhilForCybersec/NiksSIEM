Event Hub Integration

Azure Event Hub provides high-throughput streaming for data ingestion. Use it for custom applications, IoT data, or when you need to transform data before Sentinel.

When to Use Event Hub

Do you need custom transformation before data reaches Sentinel?
Yes - Need Transformation

→ Use Event Hub + Stream Analytics/Function

Event Hub receives data, Stream Analytics transforms, outputs to Log Analytics.

No Transformation Needed

→ Use Direct Connector or DCR

Simpler architecture, lower cost.
