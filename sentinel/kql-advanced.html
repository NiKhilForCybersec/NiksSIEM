KQL Advanced

Master advanced KQL techniques including time-series analysis, machine learning functions, custom functions, and query optimization for production-scale security operations.

Time Series Analysis

Time Series Functions
// make-series: Create time series data
SigninLogs
| make-series LoginCount=count() default=0 
    on TimeGenerated 
    from ago(7d) to now() step 1h
    by UserPrincipalName
| render timechart

// series_decompose: Break down trends and seasonality
SigninLogs
| make-series Count=count() on TimeGenerated from ago(30d) to now() step 1d
| extend (baseline, seasonal, trend, residual) = series_decompose(Count)
| project TimeGenerated, Count, baseline, trend

// series_outliers: Find anomalies
SigninLogs
| make-series LoginCount=count() on TimeGenerated from ago(7d) to now() step 1h
| extend outliers = series_outliers(LoginCount, 1.5)
| mv-expand TimeGenerated, LoginCount, outliers
| where outliers > 0

Machine Learning Functions

ML Functions
// basket: Find common patterns in data
SecurityEvent
| where EventID == 4688
| project Computer, Account, ProcessName
| evaluate basket(threshold=0.05)

// autocluster: Automatic pattern discovery
SigninLogs
| where ResultType != 0
| project UserPrincipalName, IPAddress, AppDisplayName, Location
| evaluate autocluster()

// diffpatterns: Compare two datasets
let Baseline = SigninLogs | where TimeGenerated between(ago(14d)..ago(7d));
let Current = SigninLogs | where TimeGenerated > ago(7d);
Baseline
| evaluate diffpatterns(Current, "Baseline", "Current")

Query Optimization

Performance Best Practices

  • Time filter first: Always filter TimeGenerated early
  • Use has over contains: has uses index, contains doesn't
  • materialize(): Cache intermediate results used multiple times
  • Avoid regex: Use has/startswith/endswith when possible
  • Limit columns: project only needed fields

Interview Questions

// Detect users with anomalous login counts
let baseline_period = 14d;
let detection_period = 1d;
let Baseline = SigninLogs
| where TimeGenerated between(ago(baseline_period)..ago(detection_period))
| summarize AvgLogins=avg(count()), StdDev=stdev(count()) by UserPrincipalName;
let Current = SigninLogs
| where TimeGenerated > ago(detection_period)
| summarize TodayLogins=count() by UserPrincipalName;
Current
| join kind=inner Baseline on UserPrincipalName
| extend ZScore = (TodayLogins - AvgLogins) / StdDev
| where ZScore > 3  // More than 3 std deviations
