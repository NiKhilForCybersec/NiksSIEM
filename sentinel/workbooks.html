Sentinel Workbooks

Workbooks are interactive dashboards in Microsoft Sentinel that visualize security data, track metrics, and support investigations. Built on Azure Monitor Workbooks with KQL.

Workbook KQL Examples

Example: Incident Trend Chart

KQL for Time Chart
SecurityIncident
| where TimeGenerated >= ago(30d)
| summarize 
    Total = count(),
    Critical = countif(Severity == "High"),
    Medium = countif(Severity == "Medium"),
    Low = countif(Severity == "Low")
  by bin(TimeGenerated, 1d)
| render timechart

Example: SOC Metrics Tiles

KQL for MTTD Calculation
SecurityIncident
| where TimeGenerated >= ago(7d)
| where Status == "Closed"
| extend 
    FirstAlertTime = todatetime(FirstActivityTimeGenerated),
    ClosedTime = todatetime(ClosedTime)
| extend 
    TimeToDetect = TimeGenerated - FirstAlertTime,
    TimeToResolve = ClosedTime - TimeGenerated
| summarize 
    MTTD_minutes = avg(TimeToDetect) / 1m,
    MTTR_minutes = avg(TimeToResolve) / 1m,
    TotalIncidents = count()
| project 
    MTTD = strcat(round(MTTD_minutes, 1), " min"),
    MTTR = strcat(round(MTTR_minutes, 1), " min"),
    TotalIncidents

Workbook Best Practices

Performance
  • Limit time ranges
  • Use summarize early
  • Project only needed fields
  • Avoid expensive joins
Usability
  • Add clear titles/descriptions
  • Use consistent colors
  • Enable drill-down
  • Group related content

Interview Questions

Key SOC metrics to include:

  • MTTD: Mean Time to Detect - time from event to alert
  • MTTR: Mean Time to Respond - time from alert to resolution
  • Incident volume: By severity, by source, trend over time
  • Analyst workload: Open incidents per analyst
  • False positive rate: Closed as FP / total closed
  • Coverage: Data sources health, ingestion volume

Design tips: Use parameters for time range, add drill-down to incident details, include trend comparisons (this week vs last week).

Azure Monitor Workbooks:

  • Rich interactivity with parameters
  • Complex layouts with steps
  • Conditional visibility
  • Edit mode for building
  • Save as templates

Azure Dashboards:

  • Simpler, pinned tiles
  • Quick overview screens
  • Pin from various Azure services
  • Less interactivity

For Sentinel, workbooks are preferred for security analysis; dashboards are better for quick executive views.
