Windows Onboarding

Onboard Windows devices to Microsoft Defender for Endpoint using various methods including Group Policy, Intune, SCCM, and local scripts.

Onboarding Methods

MethodBest ForScale

Local Script Onboarding

PowerShell Onboarding
# Download onboarding package from security.microsoft.com
# Run as Administrator
.\WindowsDefenderATPLocalOnboardingScript.cmd

# Verify onboarding
Get-MpComputerStatus | Select-Object AMRunningMode, OnboardingState

# Check connectivity
Test-NetConnection -ComputerName winatp-gw-cus.microsoft.com -Port 443
