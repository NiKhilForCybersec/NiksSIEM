Local Script Onboarding

PowerShell Onboarding Steps
# 1. Download onboarding package from security.microsoft.com
#    Settings > Endpoints > Onboarding > Download package

# 2. Run as Administrator
cd C:\Path\To\Package
.\WindowsDefenderATPLocalOnboardingScript.cmd

# 3. Verify onboarding status
Get-MpComputerStatus | Select-Object AMRunningMode, OnboardingState

# 4. Check service is running
Get-Service -Name "Sense" | Select-Object Status, StartType

# 5. Verify connectivity
$urls = @(
    "winatp-gw-cus.microsoft.com",
    "winatp-gw-eus.microsoft.com", 
    "winatp-gw-weu.microsoft.com"
)
foreach ($url in $urls) {
    Test-NetConnection -ComputerName $url -Port 443
}

# 6. Run detection test (download EICAR or run test command)
powershell.exe -NoExit -ExecutionPolicy Bypass -WindowStyle Hidden $ErrorActionPreference= 'silentlycontinue';(New-Object System.Net.WebClient).DownloadFile('http://127.0.0.1/1.exe', 'C:\test-WDATP-test\invoice.exe');Start-Process 'C:\test-WDATP-test\invoice.exe'

Group Policy Deployment

1

Download Package

Get Group Policy package from security.microsoft.com portal

2

Extract ADMX

Copy ADMX/ADML files to PolicyDefinitions folder on DC

3

Create GPO

Create new GPO, navigate to Windows Defender ATP settings

4

Configure Settings

Enable onboarding, paste workspace ID and key from portal

5

Link & Apply

Link GPO to target OU, run gpupdate on endpoints

Interview Questions

  1. Portal check: Device appears in security.microsoft.com device inventory
  2. Service check: SENSE service running (Get-Service Sense)
  3. Status check: Get-MpComputerStatus shows OnboardingState = 1
  4. Detection test: Run EICAR test file, verify alert generated
  5. Event log: Check Microsoft-Windows-SENSE/Operational for events

Port: HTTPS (443) outbound only

Key URLs:

  • *.wd.microsoft.com - Windows Defender
  • *.smartscreen.microsoft.com - SmartScreen
  • winatp-gw-*.microsoft.com - MDE gateways
  • *.blob.core.windows.net - Sample upload

Tip: Use Windows Defender ATP Client Analyzer for connectivity testing
