macOS Onboarding

Deploy Microsoft Defender for Endpoint on macOS devices using Intune, JAMF, or manual installation.

Requirements

macOS Requirements

  • macOS version: 11 (Big Sur) or later
  • Disk space: 1GB
  • System Extensions: Must approve Microsoft extensions
  • Full Disk Access: Required for complete protection
  • Network filtering: Must approve network extension

Deployment Methods

MethodBest ForApprovals
IntuneEntra ID joined MacsAuto via MDM profile
JAMF ProJAMF-managed environmentAuto via config profile
ManualTesting, small deploymentsManual approval needed

Manual Installation

1

Download Package

Get wdav.pkg from security.microsoft.com onboarding page

2

Install Package

Double-click wdav.pkg, follow installation wizard

3

Approve Extensions

System Preferences > Security > Allow Microsoft extensions

4

Grant Full Disk Access

System Preferences > Privacy > Full Disk Access > Add MDE

5

Onboard

Run onboarding script from portal

CLI Commands

mdatp CLI for macOS
# Check health
mdatp health

# Verify onboarding
mdatp health --field org_id

# Run scan
mdatp scan quick
mdatp scan full --path /Users/

# Check real-time protection
mdatp health --field real_time_protection_enabled

# Connectivity test
mdatp connectivity test

# View threats
mdatp threat list

Interview Questions

  • System Extensions blocked: User must approve in Security preferences
  • Full Disk Access missing: Leads to incomplete scanning
  • Network Extension: Required for network protection features
  • Big Sur changes: Kernel extensions deprecated, need system extensions
  • Solution: Use MDM profiles to pre-approve all required permissions
