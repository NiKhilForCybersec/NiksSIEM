MDE Troubleshooting

Diagnose and resolve common MDE issues including onboarding failures, connectivity problems, and sensor issues.

Windows Diagnostics

PowerShell Diagnostics
# Check MDE service status
Get-Service -Name "Sense" | Select-Object Status, StartType

# Check Defender status
Get-MpComputerStatus | Select-Object *

# Check onboarding state
Get-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Windows Advanced Threat Protection\Status"

# Connectivity test
Test-NetConnection -ComputerName "winatp-gw-cus.microsoft.com" -Port 443

# Check Event Logs
Get-WinEvent -LogName "Microsoft-Windows-SENSE/Operational" -MaxEvents 20

# Re-register MDE
# Stop service
Stop-Service -Name "Sense"
# Remove onboarding
reg delete "HKLM\SOFTWARE\Microsoft\Windows Advanced Threat Protection" /f
# Re-onboard with script

Common Issues

Connectivity Failures

Proxy blocking, firewall rules, SSL inspection breaking cert pinning

Sensor Not Starting

Service disabled, corrupted installation, conflicting AV

Delayed Telemetry

Network latency, high event volume, agent throttling

Onboarding Failures

Wrong package, missing prerequisites, license issues

Resolution Steps

Common Fixes
# Fix: Sensor not starting
sc config sense start= auto
net start sense

# Fix: Connectivity through proxy
# Set proxy in registry
reg add "HKLM\SOFTWARE\Microsoft\Windows Advanced Threat Protection" /v ProxyServer /t REG_SZ /d "http://proxy:8080"

# Fix: Re-onboard device
# 1. Run offboarding script
# 2. Wait 30 minutes
# 3. Run onboarding script

# Fix: Corrupted installation
# Uninstall via PowerShell
$product = Get-WmiObject Win32_Product | Where-Object { $_.Name -like "*Defender*" }
$product.Uninstall()
# Reinstall

Interview Questions

  1. Check if device is online: Can you ping/RDP to it?
  2. Verify SENSE service: Get-Service Sense - is it running?
  3. Check connectivity: Can device reach MDE URLs on port 443?
  4. Review event logs: Microsoft-Windows-SENSE/Operational
  5. Check proxy: Is proxy configured? SSL inspection breaking connection?
  6. Run Client Analyzer: Comprehensive diagnostic tool
  7. Check licensing: Is tenant license valid?
