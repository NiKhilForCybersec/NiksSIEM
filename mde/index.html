Microsoft Defender for Endpoint

Microsoft Defender for Endpoint (MDE) is an enterprise endpoint security platform providing prevention, detection, investigation, and response capabilities. Deep integration with Microsoft 365 Defender and Sentinel.

Core Capabilities

Attack Surface Reduction

ASR rules, controlled folder access, network protection, exploit protection, application control

Next-Gen Protection

Antivirus, cloud-delivered protection, behavior monitoring, PUA detection

EDR

Behavioral detection, alert investigation, advanced hunting with KQL, timeline analysis

Automated Investigation

Auto-IR, remediation actions, investigation packages, live response

Vulnerability Management

TVM integration, software inventory, security recommendations, exposure score

Threat Analytics

Threat intelligence reports, emerging threats, campaign tracking

Interview Questions

The SENSE sensor is the MDE agent that runs on endpoints:

  • Process events: Creation, termination, command lines
  • Network events: Connections, DNS queries
  • File events: Creation, modification, deletion
  • Registry events: Key/value changes
  • Logon events: Authentication attempts
  • Memory events: For advanced threat detection

Data is sent to Microsoft cloud for analysis and correlation across your tenant.

  • Connector: M365 Defender connector streams alerts/incidents to Sentinel
  • Raw telemetry: Can stream DeviceEvents, DeviceProcessEvents tables
  • Bidirectional: Sentinel incidents can trigger MDE response actions
  • Unified hunting: Query both platforms from Sentinel
  • Cost: Alerts are free, raw telemetry incurs ingestion cost
